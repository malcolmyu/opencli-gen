# opencli-gen skill

Auto-generate opencli plugins from frontend projects.

## Install

```bash
skills add opencli-gen
```

Or from GitHub:

```bash
skills add vercel-labs/agent-skills/opencli-gen
```

## Files

- `skills/opencli-gen/SKILL.md` — main entry point
- `skills/opencli-gen/discovery.md` — API discovery strategy
- `skills/opencli-gen/selection-rules.md` — endpoint selection rules
- `skills/opencli-gen/generation.md` — command generation templates
- `skills/opencli-gen/verification.md` — verification pipeline
- `skills/opencli-gen/gotchas.md` — common pitfalls
